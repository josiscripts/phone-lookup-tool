# phone-lookup-tool

Phone Lookup Tool es una aplicación de consola en Java que demuestra búsquedas rápidas de números de teléfono usando HashSet.

## funcionalidades
- almacenar números de teléfono usando HashSet
- búsqueda instantánea con contains()
- agregar nuevos números dinámicamente
- menú simple por consola

## tecnologías
- Java
- HashSet
- Scanner
- Git y GitHub

## cómo funciona
Los números de teléfono se guardan en un HashSet, lo que permite búsquedas en tiempo constante O(1).  
El usuario puede buscar o agregar números mediante un menú por consola.

## uso
1. ejecutar el programa
2. elegir una opción del menú
3. buscar o agregar números
4. salir cuando termine
